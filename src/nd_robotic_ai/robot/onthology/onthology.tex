    \begin{enumerate}
        
        \item \textbf{Bio Ontologies}
        
        Examples include:
        
        \begin{itemize}
            \item Gene Ontology
            \item NeuroLex
            \item Foundational Model of Anatomy
        \end{itemize}
        
        These ontologies are mainly concerned with:
        
        \begin{itemize}
            \item genes
            \item proteins
            \item cells
            \item biological anatomy
        \end{itemize}
        
        Relevance to this work: low.
        The present work focuses on a conceptual model of an intelligent agent or robot rather than on biological systems.
        
        Estimated confidence: about 80\%.
        
        \item \textbf{Cognitive Ontologies}
        
        Examples include:
        
        \begin{itemize}
            \item Cognitive Atlas
            \item Mental Function Ontology
        \end{itemize}
        
        These ontologies include concepts such as:
        
        \begin{itemize}
            \item Perception
            \item Attention
            \item Memory
            \item Learning
            \item Language
        \end{itemize}
        
        This is very similar to the \emph{Mind $\rightarrow$ Cognition} part of the proposed hierarchy.
        
        Relevance to this work: high.
        
        Estimated confidence: about 90\%.
        
        \item \textbf{Robotics / Agent Ontologies}
        
        Examples include:
        
        \begin{itemize}
            \item KnowRob ontology
            \item IEEE Robot Ontology
            \item SOMA ontology
        \end{itemize}
        
        These ontologies focus on concepts such as robots, sensors, actuators, agents, and environments.
    
    \end{enumerate}
